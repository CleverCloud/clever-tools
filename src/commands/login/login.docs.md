# 游닀 `clever login` command reference

## 俱뫮잺 `clever login`

Login to Clever Cloud

```bash
clever login [OPTIONS]
```

### 游뛀 Options

| Name | Description |
|------|-------------|
| `--token` `<token>` | Directly give an existing token |
| `--secret` `<secret>` | Directly give an existing secret |
