# 游닀 `clever open` command reference

## 俱뫮잺 `clever open`

Open an application in the Console

```bash
clever open [OPTIONS]
```

### 游뛀 Options

| Name | Description |
|------|-------------|
| `-a`, `--alias` `<alias>` | Short name for the application |
| `--app` `<ID_OR_NAME>` | Application to manage by its ID (or name, if unambiguous) |
