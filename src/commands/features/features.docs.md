# 📖 `clever features` command reference

## ➡️ `clever features`

Manage Clever Tools experimental features

```bash
clever features [OPTIONS]
```

## ➡️ `clever features disable`

Disable experimental features

```bash
clever features disable [OPTIONS] <FEATURES>
```

### ⚙️ Arguments

| Name | Description |
|------|-------------|
| `features` | Comma-separated list of experimental features to manage |

## ➡️ `clever features enable`

Enable experimental features

```bash
clever features enable [OPTIONS] <FEATURES>
```

### ⚙️ Arguments

| Name | Description |
|------|-------------|
| `features` | Comma-separated list of experimental features to manage |

## ➡️ `clever features info`

Display info about an experimental feature

```bash
clever features info [OPTIONS] <FEATURE>
```

### ⚙️ Arguments

| Name | Description |
|------|-------------|
| `feature` | Experimental feature to manage |

## ➡️ `clever features list`

List available experimental features

```bash
clever features list [OPTIONS]
```

### 🚩 Options

| Name | Description |
|------|-------------|
| `-F`, `--format` `<format>` | Output format (${...}) |
