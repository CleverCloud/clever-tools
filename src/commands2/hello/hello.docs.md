# 游닀 `clever hello` command reference

## 俱뫮잺 `clever hello`

Say hello to someone

```bash
clever hello [FLAGS] <NAME> <LAST-NAME>
```

### 丘뙖잺 Arguments

| Name | Description |
|------|-------------|
| `name` | Name to greet **(required)** |
| `last-name` | Last name to greet |

### 游뛀 Flags

| Name | Description |
|------|-------------|
| `--upper` | Print the greeting in uppercase |
| `--lang` `<lang>` | Language for the greeting (en, fr, es) (default: `en`) |
