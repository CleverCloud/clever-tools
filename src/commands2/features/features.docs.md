# 📖 `clever features` command reference

## ➡️ `clever features`

Manage Clever Tools experimental features

```bash
clever features
```

## ➡️ `clever features disable`

Disable experimental features

```bash
clever features disable <FEATURES>
```

### ⚙️ Arguments

| Name | Description |
|------|-------------|
| `features` | Comma-separated list of experimental features to manage **(required)** |

## ➡️ `clever features enable`

Enable experimental features

```bash
clever features enable <FEATURES>
```

### ⚙️ Arguments

| Name | Description |
|------|-------------|
| `features` | Comma-separated list of experimental features to manage **(required)** |

## ➡️ `clever features info`

Display info about an experimental feature

```bash
clever features info <FEATURE>
```

### ⚙️ Arguments

| Name | Description |
|------|-------------|
| `feature` | Experimental feature to manage **(required)** |

## ➡️ `clever features list`

List available experimental features

```bash
clever features list [FLAGS]
```

### 🚩 Flags

| Name | Description |
|------|-------------|
| `-F`, `--format` `<format>` | Output format (${...}) (default: `human`) |
