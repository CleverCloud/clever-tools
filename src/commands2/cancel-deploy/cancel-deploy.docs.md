# 游닀 `clever cancel-deploy` command reference

## 俱뫮잺 `clever cancel-deploy`

Cancel an ongoing deployment

```bash
clever cancel-deploy [FLAGS]
```

### 游뛀 Flags

| Name | Description |
|------|-------------|
| `-a`, `--alias` `<alias>` | Short name for the application |
| `--app` `<ID_OR_NAME>` | Application to manage by its ID (or name, if unambiguous) |
