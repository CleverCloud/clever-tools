# 游닀 `clever profile` command reference

## 俱뫮잺 `clever profile`

Display the profile of the current user

```bash
clever profile [FLAGS]
```

### 游뛀 Flags

| Name | Description |
|------|-------------|
| `-F`, `--format` `<format>` | Output format (${...}) (default: `human`) |
