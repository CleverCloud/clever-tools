# 游닀 `clever activity` command reference

## 俱뫮잺 `clever activity`

Show last deployments of an application

```bash
clever activity [FLAGS]
```

### 游뛀 Flags

| Name | Description |
|------|-------------|
| `-f`, `--follow` | Track new deployments in activity list |
| `--show-all` | Show all activity |
| `-F`, `--format` `<format>` | Output format (${...}) (default: `human`) |
| `-a`, `--alias` `<alias>` | Short name for the application |
| `--app` `<ID_OR_NAME>` | Application to manage by its ID (or name, if unambiguous) |
