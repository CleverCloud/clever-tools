## 俱뫮잺 `clever link` <kbd>Since 0.2.0</kbd>

Link this repo to an existing application

```bash
clever link <app-id> [options]
```

### 游닌 Arguments

| Name | Description |
|------|-------------|
| `app-id` | Application ID (or name, if unambiguous) |

### 丘뙖잺 Options

| Name | Description |
|------|-------------|
| `-a`, `--alias` `<alias>` | Short name for the application |
| `-o`, `-owner`, `--org` `<org-id\|org-name>` | Organisation to target by its ID (or name, if unambiguous) |
