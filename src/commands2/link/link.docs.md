# 游닀 `clever link` command reference

## 俱뫮잺 `clever link`

Link this repo to an existing application

```bash
clever link [FLAGS] <APP-ID>
```

### 丘뙖잺 Arguments

| Name | Description |
|------|-------------|
| `app-id` | Application ID (or name, if unambiguous) **(required)** |

### 游뛀 Flags

| Name | Description |
|------|-------------|
| `-a`, `--alias` `<alias>` | Short name for the application |
| `-o`, `-owner`, `--org` `<ID_OR_NAME>` | Organisation to target by its ID (or name, if unambiguous) |
